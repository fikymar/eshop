# Demo created while learning redux

## Technologies used

- next js
- redux-toolkit
- firebase - firestore, auth
- tailwind
- framer-motion
- eslint, prettier

## Features

- product can be added via the app
- products can be filtered by 3 categories
- products can be add to cart 
- dark mode switch
- responsive design
